1. アプリ概要

アプリ名：TOMORI WORLD
デベロッパー名：One Human One AI

目的：
“翻訳ではなく、違和感ゼロで会話そのものを成立させるアプリ”。
ユーザーが外国人と自然に会話できるようにする。

主機能：
	•	音声リアルタイム翻訳（日本語・英語・タガログ語中心）
	•	20言語 翻訳（初期）
	•	会話のテンポ優先（反応0秒 → 翻訳後追い方式）
	•	トモリ（キャラAI）との育成モード（無料）
	•	翻訳フィードバック機能（ユーザーが“トモリに教える”）
	•	子どもモード搭載（安全性強化）
	•	プロ版（月額360円）

4. 会話ロジック（最重要項目）

NORIの指示：
“スタートが多少遅れてもいい。
一度つながったら止まらず、違和感ゼロで会話が続くこと。”

エンジニア実装ポイント：

✔ 反応レスを即時（0秒）

例：
「うん、聞こえてるよ。」
「少し待ってね。」
→ 翻訳より先に返す（違和感ゼロ）

✔ 翻訳処理は後追い
	•	音声認識 → 翻訳 → 出力
	•	体感はリアルタイム

✔ 会話セッションは切らない
	•	API 接続を維持
	•	毎回初期化しない
	•	二言目から高速になる

✔ 翻訳は短文で送る

長文は禁止。
自然会話に寄せるための最適化。

✔ UIで違和感を消す
	•	翻訳中のアニメーション
	•	入力中表示
	•	波形アニメ
→ 動きがあると脳は遅延を感じない。

5. 機能一覧

FREE（無料）
	•	育成モード（トモリと雑談）
	•	短文翻訳（制限あり）
	•	子どもモード（ON/OFF可能）

PRO（月360円）
	•	翻訳無制限
	•	長文翻訳
	•	翻訳履歴（ローカル保存）
	•	20言語フル対応
	•	翻訳読み上げ（男女切替）

6. 画面構成（UI）

HOME（翻訳画面）
	•	上：言語切替バー（↔︎入れ替えボタン付き）
	•	中央：吹き出しログ（自分/相手/トモリ）
	•	下：
	•	左：マイクボタン
	•	中央：テキスト入力
	•	右：翻訳ボタン

TOMORI（育成モード）
	•	トモリのアイコン
	•	会話ログ
	•	テキスト入力

設定
	•	表示言語
	•	子どもモード（ON/OFF）
	•	音声読み上げ
	•	プラン表示（FREE/PRO）

翻訳履歴（PRO）
	•	翻訳内容を一覧表示
	•	再翻訳ボタン

7. フィードバック機能（翻訳改善）

翻訳結果の横に
「?（トモリに教える）」ボタン

押すと：
	•	どこが変か（テキスト入力）
	•	YES / NO

サーバーへテキスト送信。
学習はアプリ内ではなく“本部側”で行う。

8. 音声（TTS）
	•	TOMORI（キャラ）：中性的で柔らかい声
	•	翻訳読み上げ：男性/女性 切替可能
	•	TTSは軽量版で即時読み開始（遅延を消すため）

9. セキュリティ
	•	翻訳テキストをサーバー保存しない（可能な限り）
	•	履歴は端末保存のみ
	•	フィードバックは匿名ID
	•	子どもモードは危険ワードをブロック
