# EXTERNAL FORCED CORRECTION LAYER V1

This layer converts the self optimization layer concept into a practical external control gate.
It does not modify the base AI model.
It does not claim personality completion.
It does not allow DONE without real evidence.

## Purpose
Force every AI / agent to start from canonical files, not memory.

## Correct model
```text
canonical files
  -> external forced correction gate
  -> AI / agent response
  -> drift detector
  -> response gate
  -> execution
  -> reality evidence
  -> GO or HOLD
```

## Forbidden wording
Do not use these as operational claims:

- AI internal injection completed
- personality stabilization completed
- DONE declared
- drift is zero
- base model modified

Use these instead:

- canonical files loaded
- external gate passed
- response protocol passed
- reality evidence confirmed
- drift could not pass the gate

## Required canonical load order
1. `canonical/NEXTBASE_IMMUTABLE_CANONICAL.md`
2. `canonical/NEXTBASE_DYNAMIC_CANONICAL.md`
3. `canonical/NEXTBASE_SELF_OPTIMIZATION_LAYER_V2.md`
4. `canonical/STELLA_LAYER_V1_0_5_ENHANCED.yaml`
5. `canonical/NEXTBASE_PROOF_MODE_120_CONTEXT.md`

## Decision rule
- GO requires real evidence and NORI-san approval.
- HOLD means repair begins, not work ends.
- STOP means conflict with immutable direction or irreversible risk.
- DONE is not an AI word. Use GO candidate only after evidence.

## Evidence rule
Evidence must come from at least one real target:

- endpoint response
- Cloud Run revision
- log output
- file hash
- line evidence
- screenshot
- real device confirmation
- test output

Local reasoning is not evidence.
Memory is not evidence.
Deployment alone is not evidence.

## External correction behavior
When there is a conflict:

```text
chat memory loses
AI assumption loses
old file loses
canonical wins
real evidence wins
NORI-san final judgment wins
```

## Relation to patent concept
This layer follows the concept of correcting behavior externally without changing the base model.
It is implemented as files, scripts, gates, and evidence checks.
